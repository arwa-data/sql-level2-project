# sql-level2-project
Advanced SQL project using MySQL (filtering, aggregate functions, updates)
This project demonstrates advanced SQL concepts using MySQL.

## Features
- Filtering using WHERE and LIKE
- Aggregate functions (AVG, MAX, MIN, COUNT)
- String functions
- Numeric functions
- Data updates using UPDATE

## Tasks Implemented
- Top students (GPA > 90)
- Failed students (GPA < 60)
- Names filtering (start with A, 4 characters)
- Aggregate analysis on GPA
- Count students by level
- Distinct tracks
- Uppercase subjects
- Update gender values
- Increase GPA for low-performing students

- MySQL
